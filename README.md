# WeatherService
Проект на основе ASP.NET MVC, который позволяет загружать таблицы с данными погоды на сервер и просматривать записи таблицы вместе с фильтрацией.

## Развёртывание проекта
1. Установите зависимости.
2. Создайте или подключите базу данных для проекта. Предлагается **MSSQL Server Express**.
3. Создайте необходимые таблицы с помощью миграции. Для этого запустите команду `Update-Database` в консоли диспетчера пакетов.
4. Соберите и запустите проект. Адрес страницы будет отображён в консоли.

В корневой папке проекта содержатся SQL скрипты `TruncateData.sql` и `DeleteAllTables.sql` для очистки загруженных данных и полной очистки БД от таблиц, связанных с проектом.
